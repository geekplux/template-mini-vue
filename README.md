# template-mini-vue

> sao template for minimal Vue

## Usage

Install [SAO](https://github.com/egoist/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From git

```bash
sao geekplux/template-mini-vue my-project
```

### From npm

```bash
sao mini-vue my-project
```
